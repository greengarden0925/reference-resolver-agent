# Field mapping quick reference

## RIS
- TY: type
- AU: author, one per line
- PY: publication year
- TI: title
- T2/JO/JF: journal or secondary title
- VL: volume
- IS: issue
- SP/EP: start/end page
- DO: DOI only, no URL prefix
- UR: URL
- PB: publisher
- CY: place published
- ET: edition
- ER: end record

## EndNote tagged format
- %0: reference type
- %A: author, one per line
- %D: year
- %T: title
- %J: journal
- %V: volume
- %N: issue
- %P: pages
- %R: DOI
- %U: URL
- %I: publisher
- %7: edition
